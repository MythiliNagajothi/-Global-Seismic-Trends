# 🌍 Global Seismic Trends Dashboard

## 📌 Project Overview
The **Global Seismic Trends Dashboard** is a data analysis and visualization project developed using **SQL, Python, and Streamlit**.  
This project analyzes earthquake data to identify seismic activity patterns, magnitude trends, affected countries, and other geological insights through an interactive dashboard.

---

## 🚀 Technologies Used

- Python
- MySQL
- Streamlit
- Pandas
- Plotly / Matplotlib
- SQL

---

## 📂 Project Structure

```bash
Global-Seismic-Trends/
│
├── app.py                 # Streamlit dashboard application
├── conn.py                # Database connection file
├── queries.py             # SQL query handling
├── geology.sql            # SQL scripts and database queries
├── First_proj.ipynb       # Data Cleaning and Transformation using Pandas
├── README.md              # Project documentation
```
